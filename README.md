# Comparative-Diabetes-Prediction-Random-Forest-vs-Clinical-Baseline
A comparative study evaluating the predictive performance of a Random Forest Classifier against an ADA-based clinical risk proxy for the early detection of Type 2 Diabetes using the Pima Indians Dataset.

### **Project Description**
This project evaluates whether machine learning can provide a more accuracte and sensitive alternative to traditional clinical risk assessments for the early detection of Type 2 Diabetes in Adults. Using 768 health records from the Pima Indians Diabetes Dataset, the study compares a Random Forest Classifier against a Clinical Baseline Proxy model designed to simulate the American Diabetes Association (ADA) point-based risk guidelines.

The files attached to this repository contain the Python script used to develop the models in this research study. Also attached are the associated graphs that were generated using Matplotlib and the CSV file that contains all the data points in the dataset.
